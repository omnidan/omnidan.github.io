Arduino

As previously stated, I work a lot with atmel microcontrollers, which are used in Arduino products. I possess an Arduino UNO with an Ethernet shield and use it to prototype some of my chips. I've also created ArduinoPure, which is a framework based off the Arduino framework. The difference is, it doesn't need the Arduino IDE or any GUI running (I know Arduino released a Makefile, but that's old and only works on older versions of the Arduino framework). You can program with the Arduino framework just like you're used to now! Use "make" to compile the program and "make upload" to upload it to the microcontroller. Of course, you have to change some settings in the Makefile first (set your microcontroller, programmer, etc...). ArduinoPure also solved another problem I had with Arduino. It wouldn't upload correctly with my stk500. I think that's because Arduino is based off the stk500 protocol, so it assumes it's the Arduino-version of the stk500 protocol, but it actually is the real stk500. I've always had to compile in the IDE, take the temporary hex file and upload it manually via "avrdude". That was quite annoying. ArduinoPure solved all that for me, which is why I started it. It's still actively maintained, also, feel free to fork and help me working on it.
